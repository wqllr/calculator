什么是深度学习？

深度学习是机器学习的一个分支，它通过构建多层神经网络来学习数据的表征。深度学习在图像识别、自然语言处理、语音识别等领域取得了突破性进展。

核心概念

1. 神经网络基础

神经网络由大量的"神经元"组成，每个神经元接收输入，进行加权求和，然后通过激活函数产生输出。

2. 反向传播

反向传播是训练神经网络的核心算法，它通过计算损失函数对每个参数的梯度来更新网络权重。

3. 激活函数

常见的激活函数包括：

  • ReLU - 最常用的激活函数，计算简单效果好
  • Sigmoid - 输出0到1之间，常用于分类
  • Tanh - 输出-1到1之间，关于原点对称

实战项目：用Python实现神经网络

import numpy as np

class NeuralNetwork:
    def __init__(self, layer_sizes):
        self.weights = []
        self.biases = []
        for i in range(len(layer_sizes) - 1):
            self.weights.append(np.random.randn(layer_sizes[i], layer_sizes[i+1]) * 0.1)
            self.biases.append(np.zeros((1, layer_sizes[i+1])))
    
    def relu(self, x):
        return np.maximum(0, x)
    
    def forward(self, X):
        self.activations = [X]
        for i in range(len(self.weights)):
            z = np.dot(self.activations[-1], self.weights[i]) + self.biases[i]
            a = self.relu(z)
            self.activations.append(a)
        return a

# 创建网络
nn = NeuralNetwork([784, 128, 64, 10])
print("神经网络创建成功！")
